### 目录结构说明

- components : 存放业务公共组件
- constant : 存放业务公共常量
- pages: 存放业务页面
- services: 存放业务公共接口
- utils: 存放业务公共事件

```
单个页面目录结构参考home
```

### 路由配置

- 新增页面在 pages 中

- 当前路由
 
